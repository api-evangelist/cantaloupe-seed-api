# Cantaloupe Seed API (cantaloupe-seed-api)
The Cantaloupe Seed API connects point-of-sale and self-service retail equipment to Cantaloupe's cloud-based vending management platform, enabling cashless payment processing, real-time sales tracking, and consolidated reporting across vending, micro markets, office coffee, and unattended retail channels. Seed APIs compile transactions, fees, and payments into a single bill and expose sales analytics so operators can monitor performance, payment types, and profitability across their fleet.

## APIs

### Cantaloupe Seed API
Seed API connects a vending operator's POS and machine fleet to Cantaloupe's cashless gateway and reporting cloud. It compiles transactions, fees, and payments into a single bill across vending, micro markets, and office coffee and pantry services, and surfaces real-time sales data as comprehensive reports covering performance, payment mix, and profitability.

**Human URL:** [https://www.cantaloupe.com/solutions/services/seed-api](https://www.cantaloupe.com/solutions/services/seed-api)

#### Features

- POS integration to Cantaloupe's cashless cloud gateway
- Accepts debit/credit, mobile and digital wallets, and NFC
- Single-gateway processing for major card brands
- Consolidated billing across vending, micro markets, and office coffee
- Real-time sales data and reporting
- Payment-type analytics (cash vs cashless vs wallet)
- Profitability reporting across channels
- Fleet-wide transaction and fee reconciliation

#### Use Cases

- Enabling cashless acceptance on existing vending machines
- Unifying sales reporting across micro markets and vending
- Reconciling transactions, fees, and payouts in a single bill
- Monitoring machine and route performance in real time
- Analyzing payment type mix and channel profitability
- Integrating vending data into accounting or ERP systems
